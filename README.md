# Apps realizadas con Electron

## App Tasky

### Para levantar el proyecto

- Ejecutar 'npm install' en /course/boilerplates/tasky
- En /course/boilerplates/tasky ejecutar 'npm run start' para levantar webpack y 'npm run electron' para levantar electron
